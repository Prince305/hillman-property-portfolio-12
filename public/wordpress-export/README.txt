 HILLMAN PROPERTY - WordPress Theme Reference Files
 ==================================================
 
 These HTML/CSS files are a static reference of your React website.
 Use them to recreate the design in WordPress.
 
 FILES INCLUDED:
 - style.css - Complete stylesheet with all colors, fonts, and components
 - index.html - Homepage
 - about.html - About page
 - services.html - Services page
 - contact.html - Contact page
 - header.html - Header/Navigation snippet
 - footer.html - Footer snippet
 
 WORDPRESS IMPLEMENTATION STEPS:
 
 1. INSTALL A STARTER THEME
    - Use "Starter Theme" or "Underscores" as a base
    - Or use a page builder like Elementor/Divi
 
 2. UPLOAD YOUR LOGO
    - Upload the logo files to your WordPress media library
    - Use the same filenames referenced in the HTML
 
 3. CREATE PAGES
    - Create pages in WordPress: Home, About, Services, Contact
    - Copy the content sections from each HTML file
 
 4. SET UP MENUS
    - Create a menu with: Home, About, Services, Contact
    - Use the header.html as reference for styling
 
 5. ADD THE CSS
    - Add style.css to your WordPress theme's style.css
    - Or use "Additional CSS" in Customizer
 
 6. CONTACT FORM
    - Install Contact Form 7 or WPForms plugin
    - Style using the form classes in the CSS
 
 BRAND COLORS (HSL Format):
 - Navy: hsl(223, 47%, 11%) - #0D1A3B
 - Navy Light: hsl(223, 47%, 20%)
 - Gold: hsl(43, 96%, 56%) - #F5B800
 - Platinum: hsl(210, 15%, 91%)
 
 FONTS:
 - Primary: Montserrat (Google Fonts)

   <link href="https://fonts.googleapis.com/css2?family=Montserrat:wght@300;400;500;600;700;800&display=swap" rel="stylesheet">

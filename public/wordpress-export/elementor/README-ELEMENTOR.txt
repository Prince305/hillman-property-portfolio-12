HILLMAN PROPERTY - Elementor Template Import Guide
===================================================

WHAT'S INCLUDED:
- home-template.json      → Homepage
- about-template.json     → About page
- services-template.json  → Services page
- contact-template.json   → Contact page

HOW TO IMPORT INTO WORDPRESS + ELEMENTOR:
==========================================

STEP 1: Install Required Plugins
- Install and activate "Elementor" plugin (free version works)
- Install and activate "Elementor Pro" for advanced widgets (recommended)
- Install "Contact Form 7" or "WPForms" for the contact form

STEP 2: Create Pages
- Go to WordPress Admin → Pages → Add New
- Create 4 pages: Home, About, Services, Contact
- Set each page template to "Elementor Canvas" (full width, no theme header/footer)

STEP 3: Import Templates
- Open each page with "Edit with Elementor"
- Click the folder icon (Templates) in the bottom panel
- Click the upload arrow icon (Import)
- Select the corresponding JSON file
- Click "Insert"

STEP 4: Set Up Header & Footer
- Use Elementor Pro's Theme Builder (or a free plugin like "Header Footer Elementor")
- Create a header with your logo and navigation menu
- Create a footer matching the design (navy background, gold accents)

STEP 5: Contact Form
- The contact page has a placeholder for the form
- Install Contact Form 7 plugin
- Create a form with: Full Name, Email, Phone, Subject, Message
- Replace the placeholder text with the form shortcode

STEP 6: Upload Assets
- Upload your logo to WordPress Media Library
- Update logo references in the header/footer

STEP 7: Global Settings
- Go to Elementor → Settings → Style
- Set Default Font: Montserrat
- Set Primary Color: #0D1A3B (Navy)
- Set Secondary Color: #F5B800 (Gold)

STEP 8: Set Homepage
- Go to WordPress Admin → Settings → Reading
- Set "Your homepage displays" to "A static page"
- Select "Home" as homepage

BRAND COLORS:
- Navy: #0D1A3B
- Navy Light: #1A2E5A
- Gold: #F5B800
- Font: Montserrat (Google Fonts)
